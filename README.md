# Crawler
## Python Scrapy Practice

- start project
    
    ``scrapy startproject renrenche``
    
- gen spider
    
  ``scrapy genspider sell_car_spider``

- list spider

    ``scrapy list``
       
- run command

  ```scrapy crawl sell_car_spider```
 
- test page shell

    ``scrapy shell "https://www.renrenche.com/nj/ershouche/"``
